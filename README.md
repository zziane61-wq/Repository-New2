# منتدى احترافي (عرض توضيحي)

طريقة التشغيل:

1. افتح `index.html` في متصفح حديث.
2. أنشئ حسابًا عبر زر `تسجيل`، ثم سجل الدخول.
3. يمكنك الآن كتابة منشورات ونشرها وتحريرها وحذف مشاركاتك.

ملاحظات:
- التطبيق يعمل بالكامل على جانب العميل ويستخدم `localStorage` لحفظ المستخدمين والمشاركات.
- للاستخدام الحقيقي عبر الإنترنت تحتاج إلى خادم (API) وقاعدة بيانات.
